# 系统初衷
在班级管理的日常中，你是否曾为积分统计繁琐耗时、积分变动难以追溯、学生成果无法直观展示而烦恼？

**智能班级积分系统**，专为中小学班级管理量身打造，以**轻量化、高效化、可视化**为核心，打通班级积分管理的全流程，让老师管理更轻松，学生成长更清晰，家校沟通更便捷！

## 🎯 核心亮点，一键解锁班级管理新体验
### ✅ 一站式学生管理，高效触达每一位
告别手动录入与重复维护！系统支持学生信息的**批量新增/编辑/删除**，学生头像、班级、学号等基础信息一键归档；同时支持**头衔绑定与颜色自定义**，为优秀学生打造专属荣誉标识，让每一份努力都有迹可循、熠熠生辉。

### ✅ 智能积分规则，让奖惩有据可依
提前配置**加分/扣分规则库**（如课堂表现、作业完成、纪律管理等场景），支持规则名称、积分数值灵活修改。学生积分变动时，自动关联对应规则，无需手动核算，让积分评定公平透明、高效精准。

### ✅ 全流程积分操作，日志全程留痕
支持**单个/批量学生积分加减**，每一次操作都将自动写入**积分变动日志**，记录学生ID、操作类型、变动分数、时间等关键信息。不仅方便老师回溯操作、核对数据，更能为班级管理提供完整数据支撑，杜绝争议与遗漏。

### ✅ 可视化积分排行，成长成果一目了然
前台大屏实时展示**班级积分排行**，细分「总积分榜」「昨日加分榜」「昨日扣分榜」三大维度，支持升降序切换。学生头像、头衔、前三名专属头像框（金/银/铜）动态呈现，让优秀学生直观闪耀，也能激励全体学生共同进步，打造积极向上的班级氛围。

### ✅ 多主题自由切换，适配多元展示场景
内置**浅色毛玻璃（Light）** 等多套前台主题，管理员可在后台一键切换主题配置，前台自动加载对应风格。无论是大屏展示、教室投屏，还是家长端查看，都能匹配不同场景需求，兼顾美观与实用性。

### ✅ 数据统计与追溯，管理决策有支撑
系统整合积分日志、学生信息、规则数据，支持按时间、学生、积分类型筛选查询，支持日志导出/打印。清晰的数据呈现，让老师快速掌握班级积分整体情况，为班级管理优化、学生个性化引导提供科学依据。

## 📊 数据库支撑，安全稳定无顾虑
系统依托五大核心数据表构建完整数据闭环：
- `config` 承载系统主题配置，保障前台展示稳定切换；
- `student` 存储学生核心信息与积分基础数据，是系统的核心基石；
- `titles` 管理头衔配置，实现荣誉标识的个性化定制；
- `score_rules` 固化积分规则，保障奖惩标准的统一规范；
- `score_log` 全程记录积分变动，确保数据可追溯、可核查。

所有数据均采用`utf8mb4`编码，支持中文、表情等多格式存储，兼顾兼容性与安全性，为系统长期稳定运行筑牢基础。
